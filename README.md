# Pexels App

**Pexels App** — мобильное приложение для Android, предоставляющее доступ к каталогу фотографий через Pexels API. Приложение позволяет искать изображения по ключевым словам, просматривать популярные подборки, изучать фотографии в деталях, скачивать их на устройство и сохранять в избранное.

## Назначение приложения

Приложение ориентировано на работу с визуальным контентом и предоставляет:
- поиск изображений по пользовательскому запросу;
- просмотр коллекций и отдельных фотографий;
- удобную навигацию по каталогу изображений;
- локальное сохранение выбранных фотографий.

## Обзор экранов и функциональности

### Home Screen
Главный экран приложения.
- **Поиск:** Ввод пользовательского запроса или выбор одной из предустановленных популярных категорий.
- **Отображение контента:** Фотографии отображаются в виде ступенчатой сетки (Staggered Grid), что обеспечивает корректную работу с изображениями различных пропорций.
- **Пагинация:** Реализована бесконечная прокрутка с постраничной загрузкой данных (по 30 изображений на страницу).
- **Навигация:** Нажатие на изображение открывает экран детального просмотра.

### Details Screen
Экран детального просмотра фотографии.
- **Информация об авторе:** В верхней панели отображается имя автора изображения.
- **Просмотр изображения:** Фотография занимает основную часть экрана, поддерживается масштабирование с помощью жеста Pinch-to-Zoom.
- **Действия пользователя:**
  - **Download:** Загрузка изображения в память устройства.
  - **Bookmark:** Добавление или удаление изображения из избранного.

### Bookmarks Screen
Экран сохранённых изображений.
- **Хранение данных:** Избранные фотографии сохраняются локально в базе данных.
- **Интерфейс:** Отображение изображений в виде ступенчатой сетки с указанием имени автора под каждой карточкой.
- **Сортировка:** Список отсортирован по дате добавления — новые элементы отображаются в начале.
- **Навигация:** Переход к экрану деталей при нажатии на изображение.
- **Пагинация:** Реализована прокрутка с постраничной загрузкой данных (по 30 изображений на страницу).

## Используемые технологии

| Технология | Назначение |
|-----------|------------|
| **Kotlin** | Язык разработки |
| **Jetpack Compose** | Построение пользовательского интерфейса |
| **Clean Architecture** | Архитектурный подход |
| **Hilt** | Dependency Injection |
| **Coroutines / Flow** | Асинхронная обработка данных |
| **Retrofit** | Работа с REST API |
| **OkHttp** | HTTP-клиент |
| **Coil** | Загрузка изображений |
| **Room** | Локальная база данных сохранённых изображений |
| **SplashScreen API** | Экран запуска приложения |
| **DownloadManager** | Загрузка файлов изображений |
